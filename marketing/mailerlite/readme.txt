MAILERLITE

ImpressPages CMS plugin


INSTALL

1. Upload folder "marketing" to "ip_plugins" directory of your website.
2. Login to administration area
3. Go to developer > modules and press "install".
4. Refresh web browser (F5).
5. Now you will see a new tab Marketing -> MailerLite. Here you can upload your website newsletter subscriber to the MailerLite system.
